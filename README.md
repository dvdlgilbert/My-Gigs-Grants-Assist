My Gigs Grants Assist (My Gigs - GA)) is an app that can be used to help nonprofit organizations with grant writing. 
The app (grant application) would be able to input fields about the nonprofit (mission, goals, needs, org name, address, 
points of contact, phone numbers, websites, 5O1c3 id number (IRS Tax id), information regarding grant formats.
Secondly, the app will use AI generated recommendations on available grants and proper formatting per matching grants.
The app allows multiple (those with permissions) users to build upon or collaborate on any given grant project.
Finally, the work will be saved in a way that user can access the grant project that allows for continuing work or for 
future reference examples.
